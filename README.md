## Visualización de Datos X-Y con Flask

## Autor
Luis Alberto Viscarra Hellie

## Descripción
Este proyecto permite visualizar datos X-Y enviados desde una aplicación móvil, usando Flask.

Los datos se representan en diferentes vistas:

- Cuadrantes (página principal)
- Flechas de dirección
- Matriz LED 4x4

## Funcionamiento
La aplicación recibe datos desde un celular mediante sockets (IP y puerto), y los muestra en tiempo real en una página web.

## Tecnologías usadas
- Python (Flask)
- HTML
- CSS
